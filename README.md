# Olist E-commerce Analytics Dashboard

Интерактивный аналитический дашборд для исследования данных бразильского маркетплейса Olist. Проект демонстрирует полный цикл работы с данными: от ETL-процессов до визуализации ключевых бизнес-метрик.

## 🚀 Особенности
- **Гибридная архитектура**: Приложение автоматически переключается между PostgreSQL (локально) и SQLite (в облаке), что обеспечивает работу дашборда без сложной инфраструктуры при деплое.
- **SQL-first подход**: Все агрегации (выручка, логистика, когортный анализ) выполняются через чистые SQL-запросы.
- **Интерактивность**: Фильтры, глубокий анализ логистики и динамические графики.

## 🛠 Стек технологий
- **Backend/ETL**: Python, Pandas, SQLAlchemy.
- **Database**: PostgreSQL (prod), SQLite (demo).
- **Frontend**: Streamlit, Seaborn, Matplotlib.

## 📋 Как запустить проект
1. Клонировать репозиторий: `git clone <ссылка>`
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить ETL (для локальной работы с БД): `python load_data.py`
4. Запустить дашборд: `streamlit run app.py`
