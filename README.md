# Olist E-commerce Analytics Dashboard

Интерактивный аналитический дашборд для исследования данных бразильского маркетплейса Olist. Проект демонстрирует полный цикл работы с данными: от глубокого исследовательского анализа (EDA) до разработки масштабируемого BI-решения.

## 📊 Основные разделы
* **Финансы**: Анализ выручки и среднего чека в разрезе регионов.
* **Логистика**: Мониторинг времени доставки, расчет метрики P90 и выявление «узких мест».
* **Клиентская аналитика (Retention)**: Когортный анализ и оценка лояльности (Repeat Rate).

## 💡 Инженерный подход (SQL & Pandas)
Проект спроектирован с учетом кросс-платформенности:
* **SQL-витрина**: Сложная бизнес-логика реализована через продвинутые SQL-запросы для PostgreSQL. Оригинальные запросы вынесены в папку [`/sql_queries`](/sql_queries) для демонстрации навыков владения языком запросов.
* **Гибридная обработка**: Для обеспечения стабильности при деплое (используется SQLite) часть агрегаций реализована с помощью высокопроизводительных методов `Pandas`. Это позволяет приложению работать быстро и без ошибок в любых условиях.

## 📦 Стек технологий
* **Python**: Pandas, SQLAlchemy.
* **Database**: PostgreSQL (для локальной аналитики), SQLite (для облачного демо).
* **BI & Visualization**: Streamlit, Seaborn, Matplotlib.

## 📋 Как запустить проект
1. **Клонировать репозиторий**:
   `git clone https://github.com/Rindikt/olist-ecommerce-analytics.git`
2. **Установить зависимости**: `pip install -r requirements.txt`
3. **Запустить дашборд**: `streamlit run app.py`
